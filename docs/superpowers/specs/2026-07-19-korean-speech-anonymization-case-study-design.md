# Korean Speech Anonymization Case Study Design

## Goal

2022년 7월부터 2023년 3월까지 수행한 음성 익명화 위탁연구를, KCI 논문 성과와 함께 소개하는 한국어 포트폴리오 글로 공개한다.

## Audience and tone

- 대상: AI 보안·음성 AI 연구자와 기술 채용 담당자
- 언어: 한국어, 기술 용어는 필요한 경우에만 영어를 병기
- 형식: 결과를 먼저 보여 주고, 실험 세부 사항은 짧은 섹션으로 분리

## Page structure

1. 연구 개요: 기간, 위탁연구, KCI 게재를 짧은 메타 정보로 표시한다.
2. 문제: 방송 피치 변조가 역변환으로 복원될 수 있는 보안 취약점을 설명한다.
3. 수행 범위: 한국어 통화음성 전처리, 4개 변조 기법 비교, 파라미터 튜닝, 휴먼·자동 평가, 논문 작성을 제시한다.
4. 평가 설계: 50명 청취평가와 ECAPA-TDNN 기반 EER 평가를 설명한다.
5. 핵심 결과: VTLN 유사도 1.52, 원본 본인 음성 4.40, 타인 음성 2.72, VTLN EER 46.39%, 방송 피치 변조 EER 43.68%를 보여 준다.
6. 연구 성과: Smart Media Journal 12(2), 56–65, 2023과 DOI 링크를 제공한다.

## Visual policy

- 재사용: 프로젝트 보고서에 포함된 청취평가 화면만 사용한다. 참가자·원음·개인정보가 포함된 자산은 사용하지 않는다.
- 신규 제작: 한국어 통화음성 → 익명화 기법 → 사람·모델 평가 흐름을 보여 주는 단순 SVG 도식을 만든다.
- 결과 수치는 HTML 카드와 표로 표현해 이미지 속 작은 글씨에 의존하지 않는다.

## Scope boundaries

- 연구기간은 `2022.07 – 2023.03`으로 표기한다.
- 본인 기여는 데이터 전처리, 변조 기법·파라미터 튜닝, 청취평가 구축·운영, EER/CER 분석, KCI 논문 작성으로 표기한다.
- 원본 통화음성, 참가자 식별 정보, 외부 논문의 그림은 공개하지 않는다.
